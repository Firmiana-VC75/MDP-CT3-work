WWW as a Rube Goldberg machine

Idea

The definition of a Goldberg machine reminds me of the contraptions I saw in Tom and Jerry cartoons as a child. I want to create an art-related website where viewers can learn about paintings. At the same time, I also want to infuse the site with some fun elements.

I want to randomly display famous paintings using the Metropolitan Museum of Art's API. Afterward, the color analyzer will work to extract the primary colors from the painting.



Coding Process

The first issue I encountered while coding was with the API. Initially, I found a link on Met's open-source website. But when I copied it into my code, the webpage kept displaying error.

I verified the API link three times to ensure it was correct, only to discover the error stemmed from my incorrect use of quotation marks. The proper method is to use backticks.But a new problem has arisen.

The webpage still displays “Undefined.”



Random Function

I checked the JSON feedback. In the JSON, I can see that information is being generated, but my webpage isn't receiving it.



Make the design more reasonable

Considering that VR glasses need to be worn, I made an indentation to accommodate the nose. Also put the knobs from the top to the side.



Display

I placed this device in a public setting. In my imagination, the glasses can be used at any time.
